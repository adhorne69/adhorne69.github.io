Dernière mise à jour : 09/10/2025

Cette Politique décrit comment Jamly (“nous”) traite vos données personnelles lorsque vous utilisez l’application ou le site Jamly (le “Service”). Elle complète les CGU. Nous pouvons la mettre à jour ; nous vous informerons des changements importants via l’app.

2) Données que nous collectons

Données fournies par l’utilisateur

  • Pseudonyme, email (si fourni), avatar (si fourni)
  • Identifiant de compte

Données de jeu

  • Paramètres de parties, choix de morceaux
  • Votes/attributions, scores, classements
  • Historique de parties

Données techniques

  • Identifiants techniques d’appareil, OS, version
  • Adresse IP
  • Logs d’erreurs, crash reporting, diagnostics
  • IDFA, uniquement après autorisation via App Tracking Transparency (ATT)

Services connectés (optionnel)

  • ID de piste, artiste, album (Spotify / Apple Music / YouTube)
  • Token OAuth limité aux permissions acceptées

Paiements

Nous recevons uniquement les références de transactions et statuts d’abonnement (via Apple / Google). Nous ne traitons jamais vos données de carte bancaire.

Support

Contenu de vos messages envoyés au support.

Publicités

À ce jour, l’application n’affiche aucune publicité tierce.
Si cela change, une demande de consentement ATT sera présentée et cette Politique mise à jour.

Données sensibles

Jamly ne collecte aucune donnée sensible (RGPD).

3) Finalités et bases légales

  • Fourniture du Service : exécution du contrat
  • Sécurité / anti-fraude : intérêt légitime
  • Statistiques, amélioration : intérêt légitime ou consentement si requis
  • Communications fonctionnelles : contrat / intérêt légitime
  • Marketing : consentement ou intérêt légitime (avec droit d’opposition)
  • Obligations légales

Jamly ne vend aucune donnée personnelle.

4) Mineurs

Le Service est recommandé à partir de 13 ans.
Pour les moins de 15 ans en France, le consentement parental peut être requis.

5) Destinataires

  • Équipes internes (support, produit, technique)
  • Sous-traitants : hébergement, analytics, crash reporting, push, anti-fraude
  • Plateformes tierces (Spotify, Apple Music, YouTube), uniquement si vous connectez votre compte
  • Apple / Google pour la gestion des achats
  • Autorités ou avocats si requis par la loi

Liste détaillée des sous-traitants disponible sur demande.

6) Transferts hors UE

Lorsque certains prestataires sont hors UE, le transfert est encadré par les Clauses Contractuelles Types et mesures additionnelles si nécessaire.

7) Durées de conservation

  • Données de compte : jusqu’à suppression + 24 mois max.
  • Logs de sécurité : 12–24 mois.
  • Données de jeu : 24 mois, puis anonymisation.
  • Marketing : jusqu’au retrait du consentement ou 24 mois d’inactivité.
  • Données liées à la facturation : durée légale obligatoire.

Mesures raisonnables : chiffrement, contrôles d’accès, séparation d’environnements, sauvegardes, journalisation. Pour signaler une vulnérabilité : contact@jamly.fr

10) Cookies, SDK et consentement

  • Traceurs strictement nécessaires
  • Analytics / crash reporting
  • Publicité (si ajoutée ultérieurement, avec consentement obligatoire)

Vos préférences peuvent être modifiées dans l’app ou via les réglages d’iOS/Android.

11) Services tiers (musique)

L'application peut se connecter à des services musicaux tiers tels que Spotify, Deezer et Apple Music.
Nous utilisons uniquement les métadonnées et extraits autorisés fournis via leurs API officielles
(Spotify Web API, Deezer API, Apple Music API), conformément à leurs conditions d'utilisation.
Jamly n’héberge ni ne distribue aucun fichier audio complet.

12) Décisions automatisées

Jamly n’effectue aucun traitement automatisé produisant des effets juridiques. Les scores sont purement ludiques.
